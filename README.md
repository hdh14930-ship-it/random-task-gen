# Random Task Generator

Автор: Артём Скоробогатов

## Описание
Приложение позволяет генерировать случайные задачи из предопределенного списка. Есть возможность фильтрации задач по типу, а также добавления новых задач. Вся история генерируемых задач сохраняется в JSON файле.

## Использование
- Нажмите кнопку "Сгенерировать задачу" для получения случайной задачи.
- Используйте выпадающее меню, чтобы фильтровать задачи по типам.
- Нажмите "Добавить задачу", чтобы внести новую задачу в список. Введите задачу и тип, разделённые запятой, например: "Помыть машину,спорт".

## Возможные тесты
- Генерация нескольких задач.
- Добавление новой задачи и её отображение.
- Фильтрация по типам.
